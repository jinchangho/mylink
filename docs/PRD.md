# [PRD] 마이링크 (MyLink) - 링크트리 클론 서비스

## 1. 프로젝트 개요
- **프로젝트명**: 마이링크 (MyLink)
- **목적**: 사용자가 자신의 다양한 소셜 미디어, 웹사이트, 포트폴리오 링크를 하나의 페이지로 통합하여 관리하고 공유할 수 있는 커스텀 링크 페이지 서비스 제공.
- **대상 사용자**: 
  - 인스타그램, 틱톡 등 SNS 프로필에 하나의 링크만 게시할 수 있는 크리에이터.
  - 자신을 홍보하고 싶은 프리랜서 및 포트폴리오를 관리하는 개인.
  - 여러 채널을 운영하는 소규모 비즈니스 운영자.

---

## 2. 핵심 기능 목록 (필수 기능)

1. **사용자 인증 및 계정 관리**: 구글 계정 소셜 로그인 전용, 로그아웃.
2. **링크 관리**: 링크 추가, 수정, 삭제.
3. **자동 아이콘 생성**: 구글 파비콘 API를 사용하여 입력된 URL의 아이콘 자동 표시.

---

## 3. 기능 상세 설명

### 3.1 사용자 인증
- **구글 소셜 로그인 전용**: 별도의 이메일 가입 절차 없이 구글 계정으로 즉시 연동.
- 로그인 후 사용자별 고유 URL(예: `mylink.com/username`) 확보.

### 3.2 링크 관리 시스템
- **입력 항목**: 링크 제목(Label), 연결 URL.
- **아이콘**: 구글 파비콘 API(`https://www.google.com/s2/favicons?domain=...`)를 호출하여 사이트 아이콘을 자동 표시.
- **검증**: URL 입력 시 유효한 형식인지 체크.
