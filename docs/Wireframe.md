```text
|                +---------------------------+                |

|                                                             |

```

### 2.2 가입 중 (Sign-up/Login Flow)
- **목적**: 구글 OAuth를 통한 인증 및 초기 설정

```mermaid
sequenceDiagram
    participant User as 사용자
    participant App as 마이링크 앱
    participant Google as 구글 인증 서버
    participant DB as 데이터베이스

    User->>App: [구글 계정으로 시작하기] 클릭
    App->>Google: 인증 요청 (OAuth 2.0)
    Google-->>User: 구글 로그인 화면 표시
    User->>Google: 로그인 정보 입력 및 승인
    Google-->>App: 인증 토큰 전달
    App->>DB: 사용자 정보 확인/저장
    DB-->>App: 처리 완료
    App->>User: 관리자 대시보드로 이동
```

## 3. 화면 구성 요소 상세 설명

### 3.1 가입 전 (Landing)
- **CTA 버튼**: 구글 로그인 버튼을 중앙에 배치하여 이탈률을 최소화함.

### 3.2 가입 중 (Auth Flow)
- **OAuth 2.0**: 사용자에게 별도의 비밀번호 입력을 요구하지 않고 보안성을 확보함.

### 3.3 가입 후 (Dashboard)
- **링크 리스트**: 현재 등록된 링크들을 리스트 형태로 보여주며, 구글 파비콘 API를 통해 아이콘을 자동 표시함.
- **실시간 미리보기 (옵션)**: 우측 또는 하단에 실제 방문자가 보게 될 화면을 미리 보여주는 기능을 고려할 수 있음.
